#Build on linux

### First: Boost library
#####Debian/Ubuntu:

```
sudo apt-get install libboost-dev
sudo apt-get install libboost-system-dev libboost-coroutine-dev libboost-thread-dev

```

#### ArchLinux
```
sudo pacman -S boost
```

### Then: Make binary
```
git clone https://github.com/topcpporg/cinatra.git
cd cinatra
mkdir build
cd build
cmake ..
make
```
##### Done!
